Projects

Medicine Ordering Web App

This web application serves as an online system for ordering medications. The login screen verifies and authenticates the user, such as customer or administrator, and provides different privileges. Customers can view the medicine database, add items to their cart, and submit their order through the system.

Administrators can input, update, and delete medicine data through the system.

Cab Booking Web App

This is a simple web application for booking a cab and for calculating cab fare based on the pickup location, destination, and vehicle of choice.

A customer can input their information into the system and book their cab by clicking on the "Book a Cab" button. A confirmation notification is then displayed to the user after submitting their booking. This data is safely stored in a database and can be accessed by administrators.

Zumba Class Management Web App

Gym Solutions provides a fully functional web app made for Zumba fitness instructors to register, view and manage their students' attendance in class. The administrator and/or instructor can view a list of all registered participants as well as sub divided lists of each class session. Similarly, the administrator can schedule new class times and update session information.
