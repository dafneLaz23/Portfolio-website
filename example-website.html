Markup and Programming languages used:

HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It provides the basic building blocks of a webpage by organizing elements like headings, paragraphs, images, links, and sections.

CSS

CSS (Cascading Style Sheets) is the language used to style and design webpages. While HTML provides the structure of a website, CSS controls how it looks — including colors, fonts, spacing, layouts, and responsiveness.

Javascript

JavaScript is a programming language that adds interactivity and dynamic behavior to websites. While HTML structures content and CSS styles it, JavaScript makes a webpage responsive to user actions — like clicking buttons, submitting forms, opening menus, or updating content without refreshing the page.
